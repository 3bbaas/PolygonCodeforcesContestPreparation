# Quick Guide for ```LaTeX```

## Equations

| type    | LaTeX    |
|---------|----------|
| inline  | `$eq$`   |
| display | `$$eq$$` |

## Math Operators

| operator       | LaTeX            |            |
|----------------|------------------|------------|
| Addition       | `$a+b$`          | `a+b`      |
| Subtraction    | `$a-b$`          | `a-b`      |
| Multiplication | `$a \times b$`   | `a*b`      |
| Division       | `$\dfrac{a}{b}$` | `a/b`      |
| Sqrt           | `$\sqrt{a}$`     | $\sqrt{a}$ |
