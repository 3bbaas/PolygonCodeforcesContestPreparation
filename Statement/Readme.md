# Quick Guide for ```LaTeX```

## Equations

| type    | LaTeX    |
|---------|----------|
| inline  | `$eq$`   |
| display | `$$eq$$` |

## Math

| operator       | LaTeX                |       result       |
|----------------|----------------------|:------------------:|
| Addition       | `$a+b$`              |       $a+b$        |
| Subtraction    | `$a-b$`              |       $a-b$        |
| Multiplication | `$a \times b$`       |    $a \times b$    |
| Division       | `$\dfrac{a}{b}$`     |   $\dfrac{a}{b}$   |
| Sqrt           | `$\sqrt{a}$`         |     $\sqrt{a}$     |
| Summation      | `$$\sum_{a=0}^{b}$$` | $$\sum_{a=0}^{b}$$ |
| Dots           | `$\cdots$`           |      $\cdots$      |
| UnderScore     | `$\_$`               |        $\_$        |
| Dash           | `$\in$`              |       $\in$        |
