# Quick Guide for ```LaTeX```

## Equations

| type    | LaTeX    |
|---------|----------|
| inline  | `$eq$`   |
| display | `$$eq$$` |

## Math

| Description    | LaTeX                |       Result       |
|----------------|----------------------|:------------------:|
| Addition       | `$a+b$`              |       $a+b$        |
| Subtraction    | `$a-b$`              |       $a-b$        |
| Multiplication | `$a \times b$`       |    $a \times b$    |
| Division       | `$\dfrac{a}{b}$`     |   $\dfrac{a}{b}$   |
| Sqrt           | `$\sqrt{a}$`         |     $\sqrt{a}$     |
| Summation      | `$$\sum_{a=0}^{b}$$` | $$\sum_{a=0}^{b}$$ |
| Dots           | `$\cdots$`           |      $\cdots$      |
| Element of     | `$\in$`              |       $\in$        |
| Dash           | `---`                |         -          |
| gcd            | `$\gcd$`             |       $\gcd$       |
| mod            | `$\mod$`             |       $\mod$       |
| less than      | `$\le$`              |       $\le$        |
| greater than   | `$\ge$`              |       $\ge$        |
| subscript      | `$a_i$`              |       $a_i$        |
| superscript    | `$a^b$`              |       $a^b$        |
