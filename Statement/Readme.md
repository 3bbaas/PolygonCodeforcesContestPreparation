# Quick Guide for ```LaTeX```

## Equations

| type    | LaTeX    |
|---------|----------|
| inline  | `$eq$`   |
| display | `$$eq$$` |

## Math Operators

| operator       | LaTeX            |       result       |
|----------------|------------------|:------------------:|
| Addition       | `$a+b$`          |       $a+b$        |
| Subtraction    | `$a-b$`          |       $a-b$        |
| Multiplication | `$a \times b$`   |    $a \times b$    |
| Division       | `$\dfrac{a}{b}$` |   $\dfrac{a}{b}$   |
| Sqrt           | `$\sqrt{a}$`     |     $\sqrt{a}$     |
| Sqrt           | `$\sqrt{a}$`     | $$\sum_{a=0}^{b}$$ |
